📌 Introduction


This is a modern 3D Developer Portfolio Website designed to showcase my skills, projects, and professional experience in a highly interactive and visually appealing way. The portfolio is built with a focus on smooth animations, responsive design, and immersive user experience using 3D web technologies.



It highlights my journey as a full-stack developer and provides a structured way for recruiters and visitors to explore my work, technical expertise, and contact information.



🚀 Features


🎯 1. Interactive 3D UI


Fully interactive 3D elements using Three.js
Smooth camera movement and object animations
Immersive landing experience for users


⚡ 2. Smooth Animations


Framer Motion used for page transitions
Scroll-based animations for better engagement
Floating and dynamic UI components


💼 3. Projects Showcase


Dedicated section for all major projects
Each project includes description, tech stack, and links
Clean and organized layout for better readability


🧠 4. Skills Section


Visual representation of technical skills
Includes frontend, backend, and tools expertise
Interactive 3D skill models


📞 5. Contact System


Integrated contact form using EmailJS
Users can directly send messages from portfolio
Instant email delivery to personal inbox


🌌 6. Background Effects


Animated star background
Floating particles and 3D environment
Modern futuristic UI feel


📱 7. Responsive Design


Fully responsive across all devices
Optimized for mobile, tablet, and desktop
Clean UI/UX for all screen sizes


🛠️ Tech Stack

Frontend
React.js
Tailwind CSS
Framer Motion
🎮 3D & Animation
Three.js
React Three Fiber
React Three Drei
🔧 Backend / Services
EmailJS (for contact form)
